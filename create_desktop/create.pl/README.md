# Perl Version for Creating Infinity DE

1. Check if root permission. If not, exit.
2. Update /etc/modules.conf
3. Setup Home
4. Copy files and directories to $HOME
5. Install services
6. Copy rc files from /usr/pkg/share/examples/rc.d to /etc/rc.d
7. Update /etc/rc.conf
8. Install packages
9. Update xdm/Xresources
10. Add .xinitrc
11. Symlink .xinitrc .xsession
